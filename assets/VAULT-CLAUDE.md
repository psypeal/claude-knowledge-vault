
## Knowledge Vault

This project has a knowledge vault at `.vault/`. Use the `knowledge-vault` skill to operate it.

**Commands**: `vault init`, `vault ingest`, `vault compile`, `vault lint`, `vault cleanup`, `vault query`, `vault process`, `vault status`, `vault agent reset`

The vault is not consulted unless explicitly asked. All wiki content is maintained by Claude — do not edit `.vault/wiki/` or `.vault/agent.md` files manually.
